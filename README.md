# 海南政企项目信息聚合平台

5源采集 · 跨源合并 · IT强化标注

## 在线访问

https://你的用户名.github.io/hainan-platform/

## 功能特性

- 统计卡片：总项目数、IT项目数、跨源合并数、来源数
- IT项目专区：紫色高亮区，集中展示IT/数字化项目
- 行业分布饼图：16个行业的项目分布
- 来源统计柱图：5个数据源的贡献量
- 项目列表：支持搜索、筛选、分页

## 数据源

1. 海南省公共资源交易服务中心
2. 海南省发改委
3. 海南省财政厅
4. 海南省政府网站
5. 海南省工信厅

## 本地使用

1. 双击 `启动.bat` 启动完整版（需要 Python）
2. 或打开 `index.html` 查看静态版（无需服务器）

## 数据更新

静态版数据更新需要重新运行 `generate_data.py` 生成 `data.json`，然后重新上传到 GitHub。
